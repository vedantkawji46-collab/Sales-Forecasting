# Sales-Forecasting

## Objective 
 Predict future sales based on historical data.
 
## Dataset
Book1 dataset contains:
* Store
* Dept
* Date
* Weekly_Sales
* lS Holiday

## Steps Performed
1. Load Libraries
2.  Load Dataset
3.  Data Cleaning
4.  Feature Engineering (Promotion)
5.  Data Reduction
6.  Visualization
7.  Time Series Conversion
8.   Decomposition
9.   ARIMAX Model (Main Model)
10.  Forecasting
11.  ETS Model (Alternative Model)
12.  Train-Test Split
13.  Model Training (Both Models)
14.  Accuracy Evaluation
15.   Model Comparison
16.  Final Visualization
## Tool & Library Used
* R
* excel
* library(dplyr)
* library(ggplot2)
* library(lubridate)
* library(forecast)
* library(tseries)
* library(prophet)
